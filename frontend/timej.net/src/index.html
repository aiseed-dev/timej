aiseed-migration-kit - 組織のITを、自分の手に

公開中

Microsoft 365・業務システム・CMS/スマートフォン——三つの閉じた世界から、 git・SQL・メール・静的公開という開いた土台へ。 世界を繋ぐコードをAIが書けるようになった今、組織のIT移行は現実の選択肢です。 amig コマンドがその移行を段階ごとに支援します。

  • 脱CMSの静的化 — 取込→分類→Markdown化→生成→Pages配信
  • 様式+メール受付 — Webフォームを作らない・VBAを使わない問い合わせ設計
  • 内容はMarkdown+git — 版管理・共同編集・承認。囲い込まれない
🔓 詳しい説明を読む 💻 GitHub

開発から生まれたOSSパッケージ

公開中

サイトやアプリを作る中で「無いと困る」道具を、小さなOSSパッケージに切り出して公開しています。 Cloudflare Pagesへの配信、日本語フレンドリーなMarkdown、日本語組版、FlutterのSVG描画、組織のIT移行—— どれも実際の製品の中で毎日動いている道具です。

  • cf-publish — PythonだけでCloudflare Pagesへ配信
  • CJKフレンドリー三部作 — Markdown・組版・SVGの日本語問題を解く
  • aiseed-migration-kit — 組織のITを開いた土台へ移す amig CLI
📦 パッケージ紹介を読む 💻 GitHub

世界時計 - 世界の時間と時差

公開中

ビジネスや観光に、シンプルで使いやすい世界時計アプリ。複数のタイムゾーンを同時表示し、時差計算やリマインダー機能で国際的なコミュニケーションをサポートします。

  • 複数都市同時表示 — 世界中の時刻を一画面で確認
  • タイムゾーン対応リマインダー — 複数都市の時刻を同時通知
  • 時差早見表 — 24時間のビジネスアワーを可視化
  • iOS・Android対応 — スマホでもタブレットでも同じ操作感
  • プライバシー重視 — 個人情報を収集しない設計
