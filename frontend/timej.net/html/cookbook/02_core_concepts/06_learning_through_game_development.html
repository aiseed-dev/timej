重要概念#2-6: ゲーム開発で学ぶFlutterの基本的な考え方

ゲーム開発は、Flutterの状態管理、UI更新、ユーザーインタラクションといった核心的な概念を楽しく学ぶ最良の方法の一つです。

AI時代の今、有名なゲームなら「○○ゲームを作って」と言うだけでAIが作ってくれます。しかし、オリジナルのゲームを作ろうとすると、ゲームのロジックをAIに教える必要があります。

このレシピでは、AIにゲームロジックを教える4つの方法を学び、その過程でFlutterの基本的な考え方を深く理解します。

なぜゲーム開発で学ぶのか

ゲームが教えてくれること

【状態管理】
ゲームスコア、プレイヤー位置、敵の動き
→ 全て「状態」として管理する必要がある

【UI更新】
状態が変わったら即座に画面を更新
→ Flutterの宣言的UIの本質

【ユーザーインタラクション】
タップ、スワイプ、ドラッグ
→ イベント処理とレスポンス

【タイマーと非同期処理】
ゲームループ、アニメーション
→ Future, Stream, Timer の実践

【データ構造】
ゲームボード、アイテムリスト
→ List, Map, Set の活用

enum EnemyType {
  fast,   // 速い敵: HP 50, Speed 3.0
  tank,   // 硬い敵: HP 200, Speed 1.0
  flying, // 飛行敵: HP 80, Speed 2.0, 道を無視
}

class Enemy {
  final EnemyType type;
  double hp; // ダメージを受けて減るので非final
  final double speed;
  final bool canFly;

  Enemy._(this.type, this.hp, this.speed, this.canFly);

  // 種類ごとのステータスは factory コンストラクタで振り分ける
  factory Enemy(EnemyType type) {
    switch (type) {
      case EnemyType.fast:
        return Enemy._(type, 50, 3.0, false);
      case EnemyType.tank:
        return Enemy._(type, 200, 1.0, false);
      case EnemyType.flying:
        return Enemy._(type, 80, 2.0, true);
    }
  }
}

実装しました。飛行敵は道を無視して直線で進みます。

class Tower {

  final double damage;
  final double range;
  final double attackSpeed;
  final bool canTargetFlying;
  final double slowEffect; // 0.0-1.0

}

## これがFlutterの基本的な考え方を学ぶ理由

### 1. 状態管理の重要性

**ゲームは常に状態が変化する**

```dart
class GameState {
  // プレイヤーの状態
  int score = 0;
  int lives = 3;
  int level = 1;

  // ゲームオブジェクトの状態
  List<Enemy> enemies = [];
  List<Bullet> bullets = [];
  Player player = Player();

  // ゲームフロー状態
  GameStatus status = GameStatus.ready; // ready, playing, paused, gameOver

  // 時間の状態
  double elapsedTime = 0;
}

状態が変わったらUIを更新:

class GameWidget extends StatefulWidget {
  @override
  State<GameWidget> createState() => _GameWidgetState();
}

class _GameWidgetState extends State<GameWidget> {
  GameState gameState = GameState();

  void updateGame() {
    setState(() {
      // ゲームロジックで状態を更新
      gameState.elapsedTime += 0.016; // 約60FPS

      // 敵の移動
      for (var enemy in gameState.enemies) {
        enemy.move();
      }

      // 当たり判定
      checkCollisions();

      // スコア更新
      gameState.score += 10;
    });
  }

  @override
  Widget build(BuildContext context) {
    // 状態に基づいてUIを構築
    return Column(
      children: [
        Text('Score: ${gameState.score}'),
        Text('Lives: ${gameState.lives}'),
        GameField(enemies: gameState.enemies),
      ],
    );
  }

// GameLoopはWidgetではないのでsetStateを直接呼べない。
// 代わりにコールバック(onTick)を受け取り、UI側で画面更新につなげる
class GameLoop {
  GameLoop({required this.onTick});

  final void Function() onTick; // 毎フレーム呼ばれるコールバック
  Timer? _gameTimer;

  void start() {
    // 60FPS = 約16.67ms
    _gameTimer = Timer.periodic(
      Duration(milliseconds: 16),
      (timer) => onTick(),
    );
  }

  void stop() {
    _gameTimer?.cancel();
  }
}

class _BubblePopGameState extends State<BubblePopGame> {
  List<Bubble> bubbles = [];
  Bubble? firstTapped;
  int score = 0;

  void onBubbleTap(Bubble bubble) {
    setState(() {
      if (firstTapped == null) {
        // 1つ目のタップ
        firstTapped = bubble;
      } else if (firstTapped == bubble) {
        // 同じバブルをタップ → キャンセル
        firstTapped = null;
      } else if (firstTapped!.color == bubble.color) {
        // 同じ色 → 両方消す
        bubbles.remove(firstTapped);
        bubbles.remove(bubble);
        firstTapped = null;
        score += 10;

        // 新しいバブルを追加
        if (bubbles.length < 10) {
          bubbles.add(generateRandomBubble());
        }
      } else {
        // 違う色 → 選択をリセット
        firstTapped = bubble;
      }
    });
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Bubble Pop - Score: $score'),
      ),
      body: Container(
        child: Stack(
          children: bubbles.map((bubble) =>
            Positioned(
              left: bubble.position.dx,
              top: bubble.position.dy,
              child: GestureDetector(
                onTap: () => onBubbleTap(bubble),
                child: Container(
                  width: bubble.radius * 2,
                  height: bubble.radius * 2,
                  decoration: BoxDecoration(
                    color: bubble.color,
                    shape: BoxShape.circle,
                    border: firstTapped == bubble
                        ? Border.all(color: Colors.white, width: 3)
                        : null,
                  ),
                ),
              ),
            )
          ).toList(),
        ),
      ),
    );
  }
}

class _BubblePopGameState extends State<BubblePopGame> {
  Timer? gameTimer;
  int remainingSeconds = 60;
  bool gameOver = false;

  @override
  void initState() {
    super.initState();
    generateBubbles();
    startTimer();
  }

  void startTimer() {
    gameTimer = Timer.periodic(Duration(seconds: 1), (timer) {
      setState(() {
        if (remainingSeconds > 0) {
          remainingSeconds--;
        } else {
          gameOver = true;
          timer.cancel();
        }
      });
    });
  }

  @override
  void dispose() {
    gameTimer?.cancel();
    super.dispose();
  }

  @override
  Widget build(BuildContext context) {
    return Scaffold(
      appBar: AppBar(
        title: Text('Score: $score | Time: ${remainingSeconds}s'),
      ),
      body: Stack(
        children: [
          // ゲームフィールド
          Container(...),

          // ゲームオーバー画面
          if (gameOver)
            Center(
              child: Container(
                padding: EdgeInsets.all(20),
                color: Colors.black87,
                child: Column(
                  mainAxisSize: MainAxisSize.min,
                  children: [
                    Text(
                      'Game Over!',
                      style: TextStyle(
                        fontSize: 48,
                        color: Colors.white,
                      ),
                    ),
                    SizedBox(height: 20),
                    Text(
                      'Final Score: $score',
                      style: TextStyle(
                        fontSize: 32,
                        color: Colors.yellow,
                      ),
                    ),
                    SizedBox(height: 20),
                    ElevatedButton(
                      onPressed: resetGame,
                      child: Text('Play Again'),
                    ),
                  ],
                ),
              ),
            ),
        ],
      ),
    );
  }

  void resetGame() {
    setState(() {
      score = 0;
      remainingSeconds = 60;
      gameOver = false;
      firstTapped = null;
      generateBubbles();
      startTimer();
    });
  }
}
